## Overview

ABG (Architect Bullshit Generator) is a satirical text generator designed to emulate the inflated, pretentious, and often absurd language commonly found in architectural discourse.

The project combines:
- procedural text generation
- curated architectural jargon
- discourse parody
- minimalist interface design
- architecture culture satire

ABG generates architectural statements across three escalating levels of discourse:

- 🔵 Architecture Student
- 🟡 Pretentious Architecture Professor
- 🔴 Pritzker Jury Mode
